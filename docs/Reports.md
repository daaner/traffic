### `totals`
AUTH Bearer JWT

Вывод общих данных пользователя за период.

```php
// Транзакции для пользователя
// GET
$url = '/api/reports/totals';
// поля для запроса
$query = [
  "from" => '2023-06-01',
  "to" => '2023-07-30',
];
```

Пример ответа
```json
{
  "data": {
    "totals": {
      "profits": 2325979,
      "assets": 2593414,
      "liabilities": 267435
    },
    "days": [
      "2023-07-03",
      "2023-07-04",
      "2023-07-05"
    ],
    "assets": [
      "865404.00",
      "866892.00",
      "861118.00"
    ],
    "liabilities": [
      "67404.00",
      "123003.00",
      "77028.00"
    ],
    "profits": [
      "798000.00",
      "743889.00",
      "784090.00"
    ]
  }
}
```

Пример ошибки
```json
{
    "success": false,
    "errors": {
        "from": [
            "Данные from должны быть переданы"
        ],
        "to": [
            "Данные to должны быть переданы"
        ]
    }
}
```
